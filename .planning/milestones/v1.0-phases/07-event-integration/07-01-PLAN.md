**2. Create eventBudgetTransformers.ts** with these functions:

a) `periodToDailyAllocations(period: Period, type: DailyAllocationType): DailyTimeAllocationItem[]`
   - Map over `period.days` array (positional, index-based)
   - For each entry at index `i`: `{ date: period.from.add(i, 'day').format('YYYY-MM-DD'), hours: period.days[i], type }`
   - Filter out entries where hours <= 0
   - Return empty array if `period.days` is undefined/empty

b) `dailyAllocationsToPeriod(dailyAllocations: DailyTimeAllocationItem[], eventFrom: Dayjs, eventTo: Dayjs): Period`
   - Create a Period with `from: eventFrom, to: eventTo`
   - Build `days` array sized `eventTo.diff(eventFrom, 'day') + 1`, initialized to 0
   - For each dailyAllocation, compute index as `dayjs(item.date).diff(eventFrom, 'day')` and set `days[index] = item.hours`
   - Return the Period

c) `apiAllocationsToTimeParticipants(allocations: BudgetAllocation[], persons: Array<{uuid: string, firstname: string, lastname: string}>, eventFrom: Dayjs, eventTo: Dayjs): PersonTimeAllocation[]`
   - Filter allocations to HACK_TIME and STUDY_TIME types
   - Group by personId using Map
   - For each person in `persons`, find their hack and study allocations
   - Convert hackTimeDetails.dailyAllocations to Period via `dailyAllocationsToPeriod`, same for studyTimeDetails
   - Return `PersonTimeAllocation[]` with personId, personName (`${firstname} ${lastname}`), hackPeriod, studyPeriod (null if no allocation of that type)

d) `apiAllocationsToMoneyParticipants(allocations: BudgetAllocation[], persons: Array<{uuid: string, firstname: string, lastname: string}>): PersonMoneyAllocation[]`
   - Filter allocations to STUDY_MONEY type
   - For each person, find their money allocation
   - Return `PersonMoneyAllocation[]` with personId, personName, amount from studyMoneyDetails.amount (0 if no allocation)

e) `diffAllocations(loaded: BudgetAllocation[], currentTime: PersonTimeAllocation[], currentMoney: PersonMoneyAllocation[], eventCode: string, eventFrom: Dayjs, defaultBudgetType: EventBudgetType | null): { toCreate: Array<{type: 'hack'|'study'|'money', input: any}>, toUpdate: Array<{type: 'hack'|'study'|'money', id: string, input: any}>, toDelete: string[] }`
   - Build a map of loaded allocations by `personId+type` key
   - For each person in currentTime:
     - If hackPeriod is not null: check if loaded has HACK_TIME for this person. If yes and changed -> update with id. If no loaded -> create. Build HackTimeAllocationInput with `personId`, `eventCode`, `date: eventFrom.format('YYYY-MM-DD')`, `dailyAllocations` from `periodToDailyAllocations(hackPeriod, 'HACK')`
     - Same logic for studyPeriod with STUDY_TIME and type 'STUDY'
     - If hackPeriod is null but loaded has HACK_TIME for this person -> delete (allocation removed)
     - Same for studyPeriod null with loaded STUDY_TIME
   - For each person in currentMoney:
     - If amount > 0: check loaded STUDY_MONEY. If exists and changed -> update. If no loaded -> create. Build StudyMoneyAllocationInput with `personId`, `eventCode`, `date: eventFrom.format('YYYY-MM-DD')`, `amount`, `files: []`
     - If amount <= 0 and loaded exists -> delete
   - For loaded allocations whose personId is not in currentTime/currentMoney -> delete (participant removed)
   - Return { toCreate, toUpdate, toDelete }

f) `eventBudgetTypeToAllocationType(budgetType: EventBudgetType): 'HACK_TIME' | 'STUDY_TIME'`
   - Return `budgetType === 'HACK' ? 'HACK_TIME' : 'STUDY_TIME'`

g) `eventBudgetTypeToDailyType(budgetType: EventBudgetType): DailyAllocationType`
   - Return `budgetType` directly (EventBudgetType values match DailyAllocationType: 'HACK' and 'STUDY')

Import dayjs for date operations, import types from wirespec model and from UI components. Import EventBudgetType from mappings.ts.
