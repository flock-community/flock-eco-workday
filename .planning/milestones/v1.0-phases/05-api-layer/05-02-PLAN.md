**1. API-to-domain "consume" mapper (Spring @Component):**

```kotlin
@Component
class BudgetAllocationMapper(
    private val personService: PersonService,
```

Methods:
- `fun consumeHackTime(input: HackTimeAllocationInput, id: Long? = null): HackTimeBudgetAllocation` -- resolve person via `personService.findByUuid(UUID.fromString(input.personId.value))?.toDomain() ?: error("Cannot find person")`. Map dailyAllocations using `consumeDailyAllocation()`. Compute totalHours from `input.dailyAllocations.sumOf { it.hours }`. Parse date with `LocalDate.parse(input.date)`.
- `fun consumeStudyTime(input: StudyTimeAllocationInput, id: Long? = null): StudyTimeBudgetAllocation` -- same pattern as HackTime.
- `fun consumeStudyMoney(input: StudyMoneyAllocationInput, id: Long? = null): StudyMoneyBudgetAllocation` -- similar but amount = `BigDecimal(input.amount.toString())` (use toString() to avoid Double precision loss per research pitfall 4). Files mapped via `input.files.map { Document(name = it.name, file = UUID.fromString(it.file.value)) }`.

Private helper:
- `private fun DailyTimeAllocationItem.consume(): DailyTimeAllocation` -- maps date, hours, and type enum (DailyAllocationType.STUDY -> BudgetAllocationType.STUDY, .HACK -> .HACK).

**2. Domain-to-API "produce" extension functions (file-level, private to be used by controller):**

These should be top-level private extension functions (not inside the class), following ExpenseController.kt pattern where produce() functions are at the bottom of the controller file. - `internal fun HackTimeBudgetAllocation.produce(): BudgetAllocationApi` -- maps to unified BudgetAllocation API type with type=HACK_TIME, hackTimeDetails filled, others null. Use `id.toString()` for id, `person.uuid.toString()` for personId, `date.toString()` for date.

2. **Controller class:**
```kotlin
@RestController
class BudgetAllocationController(
    private val documentService: DocumentStorage,
    private val budgetAllocationService: BudgetAllocationService,
    private val hackTimeBudgetAllocationService: HackTimeBudgetAllocationService,
    private val studyTimeBudgetAllocationService: StudyTimeBudgetAllocationService,

    private val budgetAllocationMapper: BudgetAllocationMapper,

```

3. **GET endpoint (budgetAllocationAll):** `@PreAuthorize("hasAuthority('BudgetAllocationAuthority.READ')")`. Admin path: if eventCode provided, use `findAllByEventCode(eventCode)`; if personId provided, use `findAllByPersonUuid(UUID.fromString(personId), year ?: LocalDate.now().year)`; else return emptyList(). Non-admin path: resolve user code to person UUID via `personService.findByUserCode(authentication().name)?.toDomain() ?: error("Cannot find person for current user")`, then `findAllByPersonUuid(person.uuid, year ?: LocalDate.now().year)`. Map results with polymorphic `when(it)` dispatch to `produce()`.

4. **DELETE endpoint (budgetAllocationDeleteById):** `@PreAuthorize("hasAuthority('BudgetAllocationAuthority.WRITE')")`. Convert String id to Long via `request.path.id.toLong()`. Call `budgetAllocationService.deleteById(id)`. Return Response204 or Response404.

5. **Create/Update endpoints (6 methods):** Each `@PreAuthorize("hasAuthority('BudgetAllocationAuthority.WRITE')")`. Use mapper consume methods, then call service create/update. Map result with produce(). For updates, extract id from path as Long.

6. **File endpoints (manual, not wirespec):**
- `@PostMapping("/api/budget-allocations/files") @PreAuthorize("hasAuthority('BudgetAllocationAuthority.WRITE')")` — accept `@RequestParam("file") file: MultipartFile`, call `documentService.storeDocument(file.bytes).toResponse()`.
- `@GetMapping("/api/budget-allocations/files/{file}/{name}") @PreAuthorize("hasAuthority('BudgetAllocationAuthority.READ')")` — read document, return with correct content type using `MimeMappings.DEFAULT` + `MediaType.parseMediaType()`.

7. **Private helpers:**
- `fun authentication(): Authentication = SecurityContextHolder.getContext().authentication`
- `private fun Authentication.isAdmin(): Boolean = authorities.map { it.authority }.contains(BudgetAllocationAuthority.ADMIN.toName())`
- Use `import community.flock.eco.workday.application.utils.toDomain` for Person conversion.
