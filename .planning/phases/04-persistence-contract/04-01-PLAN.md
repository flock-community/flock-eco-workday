**BudgetAllocationEntity.kt** - Abstract base entity:
```kotlin
package community.flock.eco.workday.application.budget

@Entity
@Inheritance(strategy = InheritanceType.JOINED)
abstract class BudgetAllocationEntity(
    id: Long = 0,
    code: String = UUID.randomUUID().toString(),
    @ManyToOne(fetch = FetchType.EAGER)
    open val person: Person?,
    open val eventCode: String? = null,
    open val date: LocalDate,
    open val description: String? = null,
) : AbstractCodeEntity(id, code)
```
- Use `Person` from `community.flock.eco.workday.application.model.Person`
- Person is `@ManyToOne(fetch = FetchType.EAGER)` matching Expense/Contract pattern
- Extend `AbstractCodeEntity(id, code)` matching Contract pattern
- NO `@EntityListeners` (not needed until API layer)
- Name the class `BudgetAllocationEntity` (not `BudgetAllocation`) to avoid naming conflict with domain type
